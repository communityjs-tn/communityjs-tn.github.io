JavaScript

JavaScript is a scripting language standardized by Ecma International to be, at the first time, the client-side scripting language on the web. It's originally developed by Brendan Eich under the name of Mocha, then LiveScript.

JavaScript became an importnant language in web developing, in the raise of web animation, and client side web processing. Navigators built their own engines in order to run JavaScript, trying to innovate, and 'own the JavaScript for themselves'. But Google was the best, and won the competition by developing the V8 JavaScript Engine, so the era of JS began.
